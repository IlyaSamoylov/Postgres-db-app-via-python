# Консольное приложение для работы с базой данных postgres.

Программа представляет собой консольное приложение для работы с базой данных с помощью библиотеки psycopg2.
Поддерживает добавление, просмотр, удаление и редактирование сущностей, поиск по ключу. Управление реализуется через интерфейс, не использующий технические значения и суррогатные ключи. Классы таблиц хранятся в tables/ и наследуются от dbtable. В качестве примера созданы таблицы Collections и Exhibits. Подключение осуществляется с учетом файла конфигурации config.yaml.
